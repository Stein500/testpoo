# SecureWay Results - Plateforme de résultats scolaires sécurisée

Application Next.js 14 pour la publication et consultation sécurisée des bulletins scolaires.

## Fonctionnalités

- ✅ Landing page moderne et responsive
- ✅ Authentification sécurisée avec bcryptjs
- ✅ Sessions sécurisées avec cookies httpOnly
- ✅ Téléchargement protégé des PDF
- ✅ Interface élégante avec Tailwind CSS
- ✅ Protection reCAPTCHA v2
- ✅ Rate limiting sur les connexions
- ✅ Design adapté mobile & desktop
